# SQL Assignment - Nairobi Academy Database Creation

This repository contains my SQL queries and database operations for the Student, Subjects and Exam Results from the 'Nairobi Academy' project.

## Project Overview
This project demonstrates fundamental SQL skills including:
- Creating database tables (DDL)
- Inserting, updating, and deleting records (DML)
- Filtering data using the `WHERE` clause
- Transforming data using `CASE WHEN` statements

## Queries Included

### Q9: Exam Grade Classification
Uses `CASE WHEN` to assign grades based on student marks:
- **Distinction** → marks ≥ 80
- **Merit** → marks ≥ 60  
- **Pass** → marks ≥ 40
- **Fail** → marks < 40

### Q10: Student Level Classification
Uses `CASE WHEN` to categorize students as:
- **Senior** → Form 3 or Form 4
- **Junior** → Form 2 or Form 1

## Technologies Used
- SQL (Structured Query Language)
- DDL Commands: `CREATE`
- DML Commands: `INSERT`, `UPDATE`, `DELETE`, `SELECT`
- Filtering with `WHERE` clause
- Data transformation using `CASE WHEN`

## Key SQL Concepts Demonstrated
- Proper use of `CASE WHEN ... END` with `AS` for column naming
- Filtering using operators: `=`, `>=`, `IN`, `BETWEEN`, `LIKE`
- Difference between DDL and DML
- Data manipulation and transformation
